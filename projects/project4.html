My FPS Controller

Prototype — Game Math class

About

This is my Gaming Math final project, It maybe has not so much features, but I tried to do them as precise as possible for first year game programmer.

Project Info

  • Role: Programmer
  • Engine: Unity

Introduction

It wasn't the first time I did an FPS Controller, so I knew the basics of it, but to make it more polished, I needed to research and learn new stuff for it. Now, it's a basic FPS Controller with physical and camera movement, crouching, jumping, acceleration, and deacceleration. With the crouhcing comes the checker that prevents you from standing up if there is an obsticale above, and I added speed debug just to have it display the velocity of the character.
