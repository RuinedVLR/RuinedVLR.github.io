2D Text-Based RPG Map

Prototype / Class assignment

About

It's a simple RPG map project with a simple AI implemented as an enemy entity for my Game Programming class.

Project Info

  • Role: Developer
  • Language: C#

Implementation

It is a simple map that is reading from a .txt file with walkable and not walkable objects, and there is an enemy that is constantly following you, you need to kill it to get points and with a random chance the medkit will appear for you to pick up, the main goal is to get highest score possible. You'll need a little bit of luck to accomplish it!
