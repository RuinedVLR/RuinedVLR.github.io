Slime Slasher

Gameplay — Game Programming Class

About

My first MonoGame project in an RPG/Arcade style for Game Programming class.

Project Info

  • Language: C#
  • Editor: MonoGame

Introduction

This project is my first time using MonoGame Editor which allowed me to experiment and use more features like sprite renderes, audio, animations, and much more. Theme of this game is Arcade/RPG style that requires you to fight different slimes that will try to attack you, while you using your bow will clear all the rooms from enemies.
