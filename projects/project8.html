Introduction

In this project I learnt how to do dialogue and quest systems for the first time, also I started using scriptable objects to make both my dialogues and quests customizable and easy to set up. Theme of this game is RPG/Quest game that requires you to talk to NPCs and try to satisfy their requests to continue and progress throughout the game.
